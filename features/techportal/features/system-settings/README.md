# Tech Portal System Settings

Technician-safe portal settings.

## Planned Scope

- Technician profile summary.
- Password/session preferences when supported by shared access.
- Notification preferences.
- Device/app preferences.
- Map/provider preference if safe to expose.
- Future offline sync status and cache controls.
- Help and knowledge links.

## Data Sources

- System Settings for shared branding, maps, notifications, and access/session configuration.
- Account Admin or future staff module for technician profile/team/skill records.

## Boundaries

- This is not the full admin System Settings module.
- Do not expose global configuration, secrets, device credentials, API keys, or admin access controls here.
