# THREE3J Customer Profiling Web

React + TypeScript + Vite frontend shell integrated from TailwindAdmin template.

## Run (inside web folder)
```bash
npm install
npm run dev
```

## Required env
- `VITE_WEB_BASE_PATH` (default `/customer-profiling`)
- `VITE_API_BASE_PATH` (default `/api/customer-profiling`)
- `VITE_APP_VERSION` (default `0.1.4`)

## Mandatory pages included
- Customers List
- Customer Create/Edit
- Customer Details (Profile/Services/Audit tabs)
- Service Assignment
- Audit Logs
- API Page
- AI Prompt Page
- Updates Page
